 Standalone Python multiprocessing
==================================

`multiprocessing` is a fork of R Oudkerk's `processing` package. It's
shipped with Python 2.6 and newer versions. This standalone variant
is compatible with Python 2.4 and 2.5.

Please refer to http://www.python.org/doc/2.6/library/multiprocessing.html
for more information about the `multiprocessing` package.
